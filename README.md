# CineMatch - AI Movie Recommendation Chatbot

An AI-powered movie recommendation chatbot that helps you find movies to watch. Built with vanilla JavaScript and Claude AI.

## Live Demo

Check it out here: https://karthikpoola6-cmd.github.io/CineMatch-AiChatbot

## What It Does

- Talk to an AI about movies and get personalized recommendations
- Search by genre (sci-fi, horror, comedy, action, romance, drama, animated)
- Find movies similar to ones you already like
- Filter by time period (like "movies after 2007")
- Works without an API key too (just uses a basic movie database)

## Tech Stack

- HTML, CSS, JavaScript (no frameworks)
- Claude AI API
- GitHub Pages for hosting

## Features

**AI Mode:**
- Real conversations with memory
- Personalized recommendations based on your taste
- Understands natural language

**Basic Mode:**
- 130+ curated movies in database
- Genre and keyword matching
- Works without internet or API key
